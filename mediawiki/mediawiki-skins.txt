Skins
==========

Skins are installed in $WIKI_DIR/skins. Once installed they are enabled
in LocalSettings.php. The branch, REL1_30, should be kept in sync with
the Mediawiki software being run. Once something is installed it can be
updated with update-wiki.sh script.

git clone https://github.com/wikimedia/mediawiki-skins-CologneBlue.git
mv mediawiki-skins-CologneBlue CologneBlue
cd CologneBlue
git checkout REL1_30
cd ..
